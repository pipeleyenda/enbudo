Ejercicio 4.2

1. ¿Qué es HTML?

Es un lenguaje de marcado que se usa para estructurar páginas web.

2. ¿Qué significa HTML?

Significa HyperText Markup Language.

3. ¿Cuáles son las etiquetas básicas de HTML?

<html>, <head>, <title>, <body>.

4. ¿Qué es una etiqueta en HTML?

Es un conjunto de caracteres que definen el comienzo y fin de un elemento.

5. ¿Qué es el atributo "href"?

Es un atributo que indica la ruta a seguir en un enlace.

6. ¿Para qué sirve la etiqueta <a>?

Para crear enlaces a otras páginas.

7. ¿Qué es un hipervínculo?

Es un enlace que lleva a otra página o sitio web.

8. ¿Qué etiqueta se utiliza para insertar una imagen?

Se usa la etiqueta <img>.

9. ¿Qué atributo se utiliza para indicar la ruta de la imagen?

Se usa el atributo "src".

10. ¿Qué atributo se utiliza para indicar un texto alternativo en una imagen?

Se usa el atributo "alt".
